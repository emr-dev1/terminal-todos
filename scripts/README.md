# Terminal Todos Scripts

Simple shell scripts to start Terminal Todos.

## Setup (First Time Only)

```bash
./install.sh
```

This installs the `terminal-todos` package with all its dependencies.

## Usage

### Start Terminal Todos

```bash
./start-terminal.sh
```

Starts the terminal-based TUI application.

## Make Scripts Executable

If the scripts don't run, make them executable:

```bash
chmod +x *.sh
```
